## Summary

The **Relationships** package is the deployable DataMiner artifact for the Relationships solution. It bundles all solution components — including **GQI data sources**, **automation scripts**, **low-code apps**, and **dashboards** — into a single installable package. Deploying this package installs the complete Relationships solution on a DataMiner System, enabling users to establish, manage, and visualize relationships between DataMiner objects.
